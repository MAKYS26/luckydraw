📋 How to Use with QR Scanner:
  1. Assign one scanner per field — Scanner 1 to field 1, Scanner 2 to field 2, etc. This prevents digits mixing (e.g. 1023 + 1030 becoming 102330).
  2. Scan QR code — scanner auto-fills its field and presses Enter
  3. System instantly checks if it's a duplicate
  4. If unique: Added to database with green confirmation
  5. If duplicate: Blocked automatically with warning showing original scan number
  6. That field auto-clears after each scan for the next QR code
  7. All team members see updates in real-time
